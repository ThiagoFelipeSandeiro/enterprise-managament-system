# Setup
1. Install XAMPP.
2. Start Apache.
3. Start MySQL.
4. Open shell.
5. Run this command ```mysql.exe –h localhost –u root -p```.
6. Continue with no password.
7. Run MySQL.sql in shell.
8. Clone repo into htdocs folder in xampp dir.
